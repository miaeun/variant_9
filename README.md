# Variant 9

## Описание
Инструмент командной строки для учебного конфигурационного языка (вариант 9).  
Вход: файл (путь задаётся параметром CLI).  
Выход: XML в stdout.

### Особенности
- Используется парсер [Lark](https://github.com/lark-parser/lark).
- Поддерживается удаление многострочных комментариев вида `#= ... =#`.
- Числа: бинарные литералы `0b...`.
- Массивы: `({ ... })`.
- Строки: `"..."`.
- Объявление константы: `имя <- значение`.
- Константные выражения: `^[ ... ]` — инфиксная арифметика и функции `mod()`, `sort()`.
- Результат — структурированное XML.
